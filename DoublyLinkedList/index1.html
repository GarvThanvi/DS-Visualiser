InsertionOperations DeletionOperations
Delete the first node:
  1. Set the head pointer to the next node.
  2. Free the memory allocated to the first node.
Delete the last node:
  1. Set the tail pointer to the previous node.
  2. Free the memory allocated to the last node.
Delete a node in the middle:
  1. Set the previous node's next pointer to the next node.
  2. Set the next node's prev pointer to the previous node.
  3. Free the memory allocated to the deleted node.
